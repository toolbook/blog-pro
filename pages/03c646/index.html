# 2020.08.05 周三

多一点功利主义，少一点人文情怀

微信、抖音、微博、快手、B站，这些国民级的软件，充斥着现代人的生活，在互联网上待久了，感觉每周都会有一个热点新闻，之前的热点新闻早已无人问津了，想紧赶时代的步伐，做一个四有青年，通过这些软件来了解国家大事，来学习知识，真的是想多了……

每天24小时中，智商不是平均分配的，世上的书籍有很多，真正有用的书籍少的可怜，格拉德威尔的书我认为是必须读的，他讲的故事非常吸引人，并且是真实的事件，在生活中，有很多不经意间，就会想起格拉德威尔讲的故事，自己的行为也受他的故事所影响，他说的话对生活有正向的指导意义，最为关键的是，你能第一时间想起他讲的故事，他的作品真的是太牛了。

在过去的一天中，一定有一些行为是错误的，人在一天中很多事情是重复的，应该说是在一生之中，有很多天做的事情是大都一样的，没有什么惊喜，没有什么意外的信息，如果大多数人的生活是很平凡的，没有什么巨大的波动，那么，按照李翔的说法，那就是在"生活区"，没错，所有人的大部分时间都在“生活区”的，是不需要调用意志力的，度过一天好像没有感觉似的，平庸的人都这样，包括你自己。

做事的时机非常重要，一天中每个时间段的效益是不同的，如果在你深度睡眠时把你叫起来去做数学题或者去看电影，你一定是非常恼火的，你甚至无法理解数学题目的意思。

尽管一天的时间是24小时，如果瓜分一下就没剩下多少了，碎片化的时间用来干什么？或许应该问碎片化的时间不能用于做什么？如果2040年的你回到2020年会干什么？

睡觉时，我头脑中经常构思一个故事，那就是回到2010年，当然，我不能改变现实，但我可以改变我的记忆(多出来一条记忆)，我认为当时正在做的事情并不重要，当时正在做的事情并不能给未来带来大的改变，当时做的事情花费大量的时间，却对事情的结局带来微小的改变，甚至让事情更糟糕了，如果回到过去(思想层面上)，我一定首先改变我的体态，我会每天去跑步，让自己变得强壮，因为成长路线是非线性的，有点像微笑曲线，刚开始是非常平滑的，就算你每天都干这件事，得到的结果也是非常平庸的，其实你不必所有时间都做这件事情，能够跳出这个局面，从别的 地方找钥匙，或许更好。。。。未完

你与李笑来的生活方式是不一样的？哪些是你需要向他学习的？

明眼人都知道，牛人之所以牛，是因为他能解决你无法解决的问题，你听说过巴菲特刷抖音吗？你会认为他不刷抖音是他的遗憾吗？不会的，只要你足够的清醒，就知道刷抖音并不能获得什么技能的提示，只能获得少许的多巴胺和之后的空虚感。要向牛人学习，至少读他们写的书，先从那些经典的书籍开始读(经典有你来定)……

# 2020.08.06 周四

昨天晚上8点多(8.06)开始冥想练习(排除一切宗教因素)，深呼吸，刚开始坐着，开始身体扫描练习，过了25分钟后，感到很吃力，就开始躺下练习，这样5分钟后闹铃响了，关掉闹铃，就感到非常舒服，转眼间就11点了。 冥想练习应该在快睡觉的时候做，其他时候只是10分钟的深呼吸练习，并且，在睡觉前的练习，都应该是坐着练，这样才能有效果。加油！

# 2020.08.09 周日

“今年过节不收礼，收礼只收脑白金”这个广告词早些年经常出现的电视广告里，那是个只有电视没有手机的年代，距今已经10几年了(那年我6岁^_^,今年2020年,那我现在几岁？)，之前一直以为这广告词是个病句，到现在才知道这是用了舛互[chuǎn hù]的修辞手法。

舛互就是互相违背,互相矛盾的意思，它是一种修辞手法。

这种修辞手法就是先否定全部，再肯定一部分，或者先肯定全部，再否定一部分，以达到突出、强调后一部分的作用。举例：大家都来了，只有小明没来；万般皆下品惟有读书高。

“哦里哇，马ki打诺卡...”，这句话在抖音里非常的火，我经常在路上走着走着，满脑子里都是带土说的这句话。

每天刷抖音，并不能给自己成长带来一丝的帮助，从抖音那里学到知识也基本不可能，学到的也大多是误导，根本就不系统。

今天读了《成功，动机与目标》前三章，我想要自己的话概括一下读到的内容，你要做一件事情，如果下个月要做这件事，你更倾向于思考"你为什么要做这件事？",如果明天要这件事，你更倾向于思考“你应该怎样做这件事？”。书读完了，但是真正理解的内容还是很少，再读一遍或许非常重要，把书的中心思想自己总结一遍或许非常的重要，书的第三章我比较熟悉，就是关于“进取型思维模式”和“表现型思维模式"的区别，我在其他很多地方都看过相似的内容，对此知识还是比较了解的，有“进取型思维模式”的人更注重自己的成长，不那么在意别人的眼光，也更容易成功。。。

人和人之间的差距没那么大，就像蚂蚁之间不会有太大差异。就像桥水基金老板瑞达利欧所说的，“我阅人无数，没见过有人天赋异禀。”

大佬的话要听啊！
